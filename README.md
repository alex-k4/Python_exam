# Prompt Engineering 

## Описание на проекта

Този проект представлява **статичен уебсайт**, създаден като курсова работа по дисциплината *Изкуствен интелект*.  
Темата е **Prompt Engineering** – изкуството и техниката за правилно формулиране на заявки към AI модели с цел получаване на по-точни, полезни и структурирани отговори.

Основната цел на проекта е да:

- обясни какво е Prompt Engineering
- покаже как работят AI моделите
- демонстрира разликата между слаб и добър prompt
- представи основни техники и методи
- даде реални примери за приложение
- покаже предимства и ограничения на подхода


## Основна идея на проекта

Prompt Engineering показва, че:

> „AI не е полезен сам по себе си – полезен е начинът, по който го насочваме.“

Колкото по-ясно, структурирано и конкретно е зададено едно запитване, толкова по-добър е резултатът.

## Технологии

Проектът е изграден с:

- HTML5
- CSS3
- Vanilla JavaScript

## Функционалности

-  Responsive layout (работи на телефон и PC)
-  Dark / Light режим
-  Scroll-to-top бутон
-  Smooth navigation между секции
-  Инфографики и таблици
-  Навигационно меню
-  Образователно структурирано съдържание
