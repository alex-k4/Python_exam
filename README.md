# Python_exam

Статичен уебсайт за курсова работа по изкуствен интелект на тема prompt engineering.

Сайтът съдържа:
- увод и основни понятия
- примери и реални приложения
- предимства и ограничения
- заключение и използвани източници

Файлът `index.html` е готов за публикуване в GitHub Pages. За да бъде достъпен, качете проекта в GitHub и активирайте Pages за основната клонка.
